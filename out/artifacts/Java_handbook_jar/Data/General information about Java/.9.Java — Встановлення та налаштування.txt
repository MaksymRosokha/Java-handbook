Якщо Ви хочете зробити повне встановлення середовища для мови програмування Java, цей розділ допоможе Вам скачати і виконати установку Java на комп'ютер.
Виконайте такі кроки, щоб створити середовище. Java SE знаходиться у вільному доступі за посиланням завантажити Java. Тому завантажуєте на комп'ютер у відповідність до своєї версії операційної системи (Щоб перейти за посиланням повернітся в головне меню, і введіть цифру 3 відповідно пункту "Завантажити Java").
Виконайте вказівки, щоб завантажити, а потім запустити .eхе і почати інсталяцію Java на комп'ютер. Як тільки Ви закінчите, Вам знадобиться розпочати налаштування змінного середовища, вказавши на правильний установний каталог.

		~Windows
Припустимо, що інсталяція Java на Windows була виконана в папку «C:\Program Files\Java каталог\JDK»:
• Клацніть правою кнопкою миші на "Мій комп'ютер" і виберіть пункт "Властивості".
• Натисніть кнопку "Змінні середовища" та вкладку "Додатково".
• Тепер, змініть змінну «PATH» (шлях), щоб вона також містила шлях до файлу, що виконується. Наприклад, якщо шлях в даний час встановлений в «C:\WINDOWS\system32», то змініть його наступним чином: з «C:\WINDOWS\system32» на «C:\Program Files\Java\JDK\Bin».

		~Linux, UNIX, Solaris, FreeBSD
Встановіть змінне середовище «PATH», щоб воно містити шлях до файлів, які були встановлені на комп'ютер раніше. Зверніться до документації вашої оболонки, якщо у Вас є проблема під час встановлення.
Наприклад, якщо Ви працюєте в оболонці, то Ви повинні додати наступний рядок в кінці «.bashrc: export PATH=/path/to/java:$PATH».
