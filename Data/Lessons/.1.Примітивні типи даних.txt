Є вісім примітивних типів даних, що підтримуються Java. Основні типи даних визначені мовою та названі за ключовим словом. Тепер подивимося в деталях ці вісім базових типів даних існуючих у мові програмування Java.


	~Тип byte
•Тип даних byte є 8-розрядним цілим цілим числом.
•Мінімальна величина дорівнює -128 (-27).
•Максимальне значення дорівнює 127 (включно) (27-1).
•За замовчуванням – 0.
•byte призначений для економії місця у великих масивах, головним чином замість цілих чисел, оскільки byte у чотири рази менше, ніж int.
•Приклад:
byte a = 100;
byte b = -50;


	~Тип short
•Тип даних short є 16-розрядним цілим цілим числом.
•Мінімальне значення дорівнює -32768 (-215).
•Максимальна величина дорівнює 32767 (включно) (215-1).
•Тип short Java може також використовуватися для економії пам'яті як byte. Розмір short у 2 рази менший, ніж int.
•За замовчуванням – 0.
•Приклад:
short s = 10000;
short r = -20000;


	~Тип int
•У мові Java тип даних int є 32-розрядним цілим знаковим числом.
•Мінімальний розмір int - 2147483648 (-231).
•Максимальна величина дорівнює 2147483647 (включно) (231-1).
•Тип int зазвичай використовується для цілих значень. Якщо немає стурбованості щодо пам'яті.
•За замовчуванням 0.
•Приклад:
int a = 100 000;
int b = -200000;


	~Тип long
•Тип даних long є 64-розрядним цілим цілим числом.
•Мінімальне значення дорівнює - 9,223,372,036,854,775,808 (-263).
•Максимальна величина - 9,223,372,036,854,775,807 (включно). (263-1).
•В Java Застосовується коли потрібно більш широкий діапазон, ніж int.
•За замовчуванням – 0L.
•Приклад:
long a = 100000L;
long b = -200000L;


	~Тип float
•Тип даних float є c одинарною точністю 32-бітний IEEE 754 з плаваючою точкою.
•Тип float використовується головним чином для збереження пам'яті у великих масивах чисел із плаваючою точкою.
•За замовчуванням 0.0f.
•Тип float ніколи не повинен застосовуватися для точного значення, наприклад, валюти.
•Приклад:
float f1 = 234.5f;


	~Тип double
•Тип даних double є c подвійною точністю 64-бітний IEEE 754 з плаваючою точкою.
•Зазвичай використовується для десяткових значень.
•Тип double ніколи не застосовується для точного значення, наприклад, валюти.
•За замовчуванням 0.0d.
•Приклад:
double d1 = 123.4;


	~Тип boolean
•Тип даних boolean є одним бітом інформації.
•Існує лише два можливі значення: true і false.
•Призначений для простих ознак, які дають змогу відстежувати умови true або false.
•За замовчуванням – false.
•Приклад:
boolean one=true;


	~Тип char
•Тип даних char є одним 16-розрядним символом Юнікод.
•Мінімальна величина дорівнює «u» (або 0).
•Максимальна величина дорівнює "\ffff" (або 65535 включно).
•Java char потрібен для зберігання будь-якого символу.
•Приклад:
char letterA = 'A';
